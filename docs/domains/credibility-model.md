# 신뢰도 모델

## 개요

신뢰도 모델은 웹 페이지의 주장을 분석하여 각 주장의 진위 여부와 전체 페이지의 신뢰도를 평가하는 기준을 정의합니다. 이 문서는 진위 판정 기준, 점수 산정 알고리즘, 근거 가중치, 출처 신뢰도를 다룹니다.

## 진위 판정 기준 (Verdict Criteria)

### 판정 유형

| 판정 | 정의 | 기준 | 신뢰도 범위 |
|------|------|------|-------------|
| `true` | 주장이 사실로 확인됨 | 복수의 신뢰할 수 있는 출처에서 확인 | 0.8 - 1.0 |
| `false` | 주장이 사실이 아닌 것으로 확인됨 | 신뢰할 수 있는 출처에서 반박 확인 | 0.8 - 1.0 |
| `mixed` | 주장이 일부 사실이고 일부 거짓임 | 일부는 사실, 일부는 거짓으로 확인 | 0.6 - 0.8 |
| `unverifiable` | 주장의 진위를 판단할 수 없음 | 주관적 표현, 검증 불가능한 내용 | 0.0 - 0.5 |

### 판정 결정 알고리즘

```
1. 주장 분석
   │
   ├── 객관적 사실 주장인가?
   │   ├── 예 → 2번继续
   │   └── 아니오 → unverifiable
   │
   ├── 출처 확인 가능 여부
   │   ├── 가능 → 3번继续
   │   └── 불가능 → unverifiable
   │
   ├── 출처 신뢰도 평가
   │   ├── 높음 (0.7 이상) → 4번继续
   │   ├── 중간 (0.4 - 0.7) → 조건부 판정
   │   └── 낮음 (0.4 미만) → unverifiable
   │
   └── 출처 간 일치 여부
       ├── 완전 일치 → true/false (출처 방향에 따라)
       ├── 부분 일치 → mixed
       └── 불일치 → unverifiable
```

### 판정 세부 기준

#### `true` 판정 기준

1. **출처 요건:** 최소 2개 이상의 독립적 출처에서 확인
2. **출처 신뢰도:** 각 출처의 신뢰도 0.7 이상
3. **일치성:** 출처 간 내용 일치
4. **시의성:** 출처가 최신 자료 (최근 2년 이내)

#### `false` 판정 기준

1. **출처 요건:** 최소 1개 이상의 신뢰할 수 있는 출처에서 반박
2. **출처 신뢰도:** 반박 출처의 신뢰도 0.8 이상
3. **명확성:** 반박 내용이 명확하고 구체적
4. **대체 설명:** 거짓으로 판명된 주장에 대한 대체 설명 존재

#### `mixed` 판정 기준

1. **부분 사실:** 주장의 일부 요소가 사실이고 일부가 거짓
2. **맥락 의존성:** 맥락에 따라 진위가 달라지는 경우
3. **시점 차이:** 과거에는 사실이었으나 현재는 아닌 경우
4. **과장 또는 축소:** 사실을 과장하거나 축소한 경우

#### `unverifiable` 판정 기준

1. **주관적 표현:** "최고", "가장 좋다" 등 주관적 의견
2. **검증 불가능:** 검증할 수 있는 방법이 없는 내용
3. **출처 부재:** 확인할 수 있는 출처가 없는 경우
4. **모호한 표현:** 구체적이지 않고 모호한 주장

## 점수 산정 알고리즘 (Score Calculation)

### 전체 신뢰도 점수 (Overall Score)

전체 페이지의 신뢰도를 0-100 범위로 나타내는 점수입니다.

#### 산정 공식

```
전체 점수 = (기본 점수 + 주장 보정 점수) × 출처 가중치

기본 점수 = 50점 (중립적 시작점)
```

#### 주장 보정 점수

```
주장 보정 점수 = Σ (주장별 보정값) / 주장 수

주장별 보정값:
- true 판정: +20점
- false 판정: -20점
- mixed 판정: ±5점 (출처 신뢰도에 따라)
- unverifiable 판정: 0점 (중립)
```

#### 출처 가중치

```
출처 가중치 = 1.0 + (평균 출처 신뢰도 - 0.5) × 0.2

평균 출처 신뢰도 = Σ (출처 신뢰도) / 출처 수
```

#### 점수 범위와 등급

| 점수 범위 | 등급 | 색상 | 설명 |
|-----------|------|------|------|
| 80 - 100 | 매우 높음 | 🟢 녹색 | 높은 신뢰도. 대부분의 주장이 사실로 확인됨 |
| 60 - 79 | 높음 | 🔵 파랑 | 양호한 신뢰도. 대부분의 주장이 사실이나 일부 주의 필요 |
| 40 - 59 | 보통 | 🟡 노랑 | 보통 수준. 주장의 진위가 혼재하거나 검증 불가 |
| 20 - 39 | 낮음 | 🟠 주황 | 낮은 신뢰도. 거짓 주장이 상당수 포함 |
| 0 - 19 | 매우 낮음 | 🔴 빨강 | 매우 낮은 신뢰도. 대부분의 주장이 거짓으로 확인 |

### 개별 주장 신뢰도 (Claim Confidence)

각 주장에 대한 LLM의 판정 신뢰도를 0-1 범위로 나타냅니다.

#### 산정 기준

| 신뢰도 범위 | 기준 | 설명 |
|-------------|------|------|
| 0.9 - 1.0 | 매우 높음 | 출처가 명확하고 일치함 |
| 0.7 - 0.89 | 높음 | 출처가 존재하나 약간의 불확실성 존재 |
| 0.5 - 0.69 | 보통 | 출처가 제한적이거나 부분적 일치 |
| 0.3 - 0.49 | 낮음 | 출처가 불분명하거나 모순적 |
| 0.0 - 0.29 | 매우 낮음 | 출처가 없거나 주장 자체가 모호함 |

## 근거 가중치 (Reasoning Weight)

근거의 설득력을 결정하는 요소에 대한 가중치입니다.

### 가중치 요소

| 요소 | 가중치 | 설명 |
|------|--------|------|
| **출처 수** | 0.3 | 인용된 출처의 수. 많을수록 높은 가중치 |
| **출처 신뢰도** | 0.35 | 출처의 신뢰도. 높을수록 높은 가중치 |
| **출처 일치성** | 0.2 | 출처 간 내용 일치 여부. 일치할수록 높은 가중치 |
| **시의성** | 0.15 | 출처의 최신성. 최근일수록 높은 가중치 |

### 근거 강도 계산

```
근거 강도 = (출처 수 가중치 × 0.3) + (출처 신뢰도 × 0.35) + (출처 일치성 × 0.2) + (시의성 × 0.15)

출처 수 가중치 = min(출처 수 / 3, 1.0)  // 3개 이상이면 만점
출처 일치성 = 일치하는 출처 수 / 전체 출처 수
시의성 = max(0, 1 - (현재 연도 - 출처 연도) / 5)  // 5년 이전이면 0
```

### 근거 강도 등급

| 강도 | 범위 | 설명 |
|------|------|------|
| 강함 | 0.8 - 1.0 | 출처가 풍부하고 신뢰할 수 있음 |
| 보통 | 0.5 - 0.79 | 출처가 존재하나 약간의 부족함 |
| 약함 | 0.2 - 0.49 | 출처가 제한적이거나 불분명 |
| 없음 | 0.0 - 0.19 | 출처가 없거나 근거가 부족 |

## 출처 신뢰도 (Source Reliability)

### 출처 유형별 기본 신뢰도

| 출처 유형 | 기본 신뢰도 | 설명 |
|-----------|-------------|------|
| `academic` | 0.9 | 학술 논문, 연구 기관, 동료 평가를 거친 자료 |
| `government` | 0.85 | 정부 기관, 공식 발표, 공공 데이터 |
| `reference` | 0.8 | 백과사전, 사전, 공식 문서 |
| `news` | 0.7 | 뉴스 매체, 기자 작성 기사 |
| `other` | 0.5 | 기타 출처, 블로그, 소셜 미디어 등 |

### 출처 신뢰도 조정 요소

기본 신뢰도에 다음 요소를 적용하여 최종 신뢰도를 계산합니다.

| 요소 | 조정값 | 설명 |
|------|--------|------|
| **독립성** | ±0.1 | 출처가 독립적인지, 단체/기업 소속인지 |
| **편향성** | ±0.1 | 출처의 정치적/이념적 편향성 |
| **시의성** | ±0.1 | 자료의 최신성 |
| **전문성** | ±0.1 | 해당 분야 전문성 |
| **수정 이력** | ±0.05 | 정정 기사 또는 수정 이력 존재 여부 |

### 최종 신뢰도 계산

```
최종 신뢰도 = 기본 신뢰도 + Σ (조정값)

단, 최종 신뢰도는 0.0 - 1.0 범위로 제한
```

## 신뢰도 시각화

### 점수 표시

- **전체 점수:** 원형 프로그레스 바로 표시
- **색상:** 점수에 따라 녹색→파랑→노랑→주황→빨강으로 그라데이션
- **숫자:** 중앙에 점수 표시 (0-100)

### 주장별 표시

- **진위 판정:** 색상으로 구분
  - `true`: 🟢 녹색
  - `false`: 🔴 빨강
  - `mixed`: 🟡 노랑
  - `unverifiable`: ⚪ 회색
- **신뢰도:** 막대 그래프로 표시 (0-1)
- **근거:** 접이식 영역으로 표시

### 등급 표시

| 등급 | 표시 |
|------|------|
| 매우 높음 | ⭐⭐⭐⭐⭐ |
| 높음 | ⭐⭐⭐⭐ |
| 보통 | ⭐⭐⭐ |
| 낮음 | ⭐⭐ |
| 매우 낮음 | ⭐ |

## 제한 사항

### 이 모델의 한계

1. **LLM 한계:** LLM은 최신 정보를 알지 못할 수 있음
2. **출처 편향:** 출처 신뢰도는 주관적 판단이 포함될 수 있음
3. **문화적 차이:** 동일한 주장이 문화에 따라 다르게 해석될 수 있음
4. **맥락 의존성:** 맥락에 따라 진위가 달라지는 주장 처리 어려움

### 개선 가능 영역

1. **실시간 출처 확인:** 웹 검색을 통한 실시간 출처 확인
2. **사용자 피드백:** 사용자의 판정 피드백을 통한 모델 개선
3. **다국어 지원:** 여러 언어의 출처 확인
4. **도메인 특화:** 뉴스, 과학, 의료 등 도메인별 특화 모델

# 도메인 문서

## 개요

이 디렉토리에는 Web Fact Checker의 핵심 도메인 로직에 대한 문서가 포함되어 있습니다. 도메인 문서는 시스템의 비즈니스 규칙과 핵심 개념을 정의합니다.

## 핵심 도메인 개념

### 텍스트 추출 도메인

- **대상**: 웹 페이지 DOM
- **입력**: 페이지 URL, DOM 트리
- **출력**: 추출된 본문 텍스트, 제목
- **핵심 로직**: readability 알고리즘을 통한 본문 영역 탐지

### 분석 파이프라인 도메인

- **대상**: 추출된 텍스트
- **입력**: 텍스트, 분석 설정
- **출력**: 분석 결과 (주장 목록, 점수, 요약)
- **핵심 로직**: LLM API 호출, 응답 파싱, 결과 가공

### 신뢰도 모델 도메인

- **대상**: 분석 결과
- **입력**: 주장 목록, 진위 판정, 근거
- **출력**: 종합 신뢰도 점수, 등급
- **핵심 로직**: 점수 산정 알고리즘, 가중치 계산

### Provider 추상화 도메인

- **대상**: 외부 LLM API
- **입력**: API 요청
- **출력**: 표준화된 분석 응답
- **핵심 로직**: API 호출, 응답 변환, 에러 처리

### 캐싱 관리 도메인

- **대상**: 분석 결과
- **입력**: 텍스트 해시, 분석 결과
- **출력**: 캐시된 결과 또는 캐시 미스
- **핵심 로직**: 저장, 조회, 만료 관리

### 설정 관리 도메인

- **대상**: 사용자 설정
- **입력**: API 키, Provider 선택, 언어 설정
- **출력**: 활성화된 설정
- **핵심 로직**: 저장, 조회, 유효성 검증
